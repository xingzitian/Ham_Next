# 鸿蒙业余无线电工具箱
![License](https://img.shields.io/badge/License-GPLv3-blue.svg)
![HarmonyOS API](https://img.shields.io/badge/HarmonyOS%20API-6.0.0(20)-brightgreen)
![Test Device](https://img.shields.io/badge/Test%20Device-Mate60-success)

一款基于**鸿蒙原生开发**的业余无线电综合工具箱，主打**工具全包揽**，一站式解决鸿蒙系统下业余无线电爱好者的使用需求，助力业余无线电走进鸿蒙生态！

经过半年开发与华为应用市场上架沟通，暂未完成上架，现开源项目供广大爱好者使用、交流与二次开发，共同推动鸿蒙与业余无线电领域的发展。

## 项目介绍
本项目为鸿蒙6.0.0(20) API编译开发的原生应用，经Mate60真机测试运行稳定，复刻并拓展了业余无线电领域经典工具的核心功能，解决了鸿蒙系统无专属业余无线电工具的痛点，实现**一个程序走天下**的使用体验，涵盖卫星追踪、日志托管、电码训练、慢扫电视等全场景功能，是鸿蒙平台业余无线电爱好者的必备工具。

## 核心功能
### 🌌 业余卫星搜索与追踪
- 内置丰富卫星资源库，支持NOAA气象卫星、ISS等经典卫星检索与添加
- 精准计算卫星过境时间、轨迹，实时定位卫星方位，支持手机姿态感应对准卫星
- 告别跨平台安装的繁琐，鸿蒙端原生体验打卫星

### 📚 业余无线电基础科普
- 整合业余无线电入门基础知识，从萌新到入门的快速学习通道
- 支持好友分享学习，一起解锁业余无线电的乐趣

### 📝 云端日志托管
- 电子化存储业余无线电操作日志，支持呼号、时间等多维度检索
- 日志数据云端托管，多端同步（后续迭代），真正实现一个程序走天下

### 🔢 摩尔斯电码全工具链
- 文字与摩尔斯电码互转，支持电码转音频播放
- 创新闪光灯联动，不便发声场景下可通过闪光灯实现摩尔斯电码求救/通信
- 内置电码听力训练、手动输入训练模块，轻松练就摩尔斯电码技能

### 📸 SSTV慢扫描电视发送
- 支持MartinMone模式的SSTV慢扫描电视图像编码与发送
- 可与好友互传图片，体验业余无线电的趣味通信（**接收功能暂未完成，持续开发中**）

## 特色亮点

#### 食用方法
1. 下载项目源码至本地
2. 使用DevEco Studio打开项目，选择鸿蒙6.0.0(20) API编译
3. 连接鸿蒙6.0+设备，直接编译运行或生成hap包安装
4. 安装后打开应用，可直接使用基础功能，卫星数据可手动拉取更新

您可以使用[Auto-installer](https://github.com/likuai2010/auto-installer/)或[DevEcho Testing](https://developer.huawei.com/consumer/cn/deveco-testing/)进行安装。

> [!IMPORTANT]
> 华为的签名服务器屏蔽了非中国大陆的IP地址，若要在非中国大陆的国家/地区为HarmonyOS NEXT侧载软件，请使用代理等方式使用中国大陆的IP地址。

> [!NOTE]
> 在HarmonyOS NEXT使用自签名的方式进行侧载的App默认只有14天的有效期，进行[开发者实名认证](https://developer.huawei.com/consumer/cn/verified/enrollment)后即可将有效期提升至180天。
